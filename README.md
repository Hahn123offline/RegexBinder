# RegexBinder
一个开发辅助工具  
### 主要功能：
全局单例。使用bind(正则, 函数)或bind(正则, 对象地址, 成员函数地址)轻松绑定，随后可以通过正则表达式触发函数的执行。  
目前支持带循环的interactive_run以及单次run指令，简单的指令查询、错误输入提示等功能。  
只支持void类型以及lambda  

### 特点
完全解耦的调试工具，允许不修改任何借口的情况下，在程序任意位置进行函数调用、参数传递。（某种很异形的信号？）  
能降低一点代码量，低成本测试各种奇怪小想法。  
适合低频次的信号发送，因为它性能实际上不好。并且未来的命令链系统能很好地操作多个模块

### 未来打算：
加入命令链进行链式调用。  
使用变量系统获取返回值。  
支持queue队列以及run_queue顺序执行命令。  
支持条件执行，允许命令之间相互控制队列中的生存周期。
